2.Add Axis2 jars to Eclipse

3.Create Dynamic web project.

5. Convert the java class into webservice using  Axis2 implementation class.
file-->other-->webservices-->>webservice-->deploy tomcat
6. Test wheather the service is deployed in the server or not.
